# 作者简介

>>Avi Wigderson教授是普林斯顿大学高等研究院著名的数学家，计算机科学家，美国科学院院士, 美国人文与科学院士，获得了计算机和数学界几乎所有的大奖，例如哥德尔奖、高德纳奖和奈望林纳奖等最高荣誉奖，还是阿里巴巴达摩院学术委员会的十位顶尖科学家之一。从事随机计算、密码学、电路复杂度、证明复杂度、并计算等计算机科学基础领域。
